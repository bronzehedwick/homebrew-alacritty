## Requirements

## Installation
Since this formula isn't part of Homebrew, add it as tap first:
```
$ brew tap yuu/alacritty
```

To install the latest, potentially unstable, version of Neovim from git HEAD:
```
$ brew install --HEAD alacritty
```
